

# Automation Exercise - Manual Testing Project

## Project Overview
This repository contains a comprehensive manual testing suite for the [Automation Exercise](https://automationexercise.com/) e-commerce platform. The project demonstrates end-to-end quality assurance processes, from test planning and scenario design to defect reporting and final validation.

## Deliverables
- **Test Plan:** Detailed strategy, scope, and environment setup.
- **Test Suite:** 48 test cases covering Registration, Login, Product Search, Shopping Cart, and Checkout.
- **Bug Report:** Documented defects with severity levels and reproduction steps.

## Key Modules Tested
- **Registration & Account Management:** Validated user onboarding, including negative testing for invalid data and basic security (SQLi).
- **Authentication:** Verified Login/Logout flows and session management across multiple devices.
- **Product Discovery:** Tested search accuracy and category filtering.
- **Checkout Workflow:** Validated cart logic, address verification, and mock payment processing.

## Tools & Environment
- **Platform:** Web (Desktop & Mobile)
- **Browsers:** Google Chrome, Microsoft Edge
- **Documentation:** Microsoft Excel, Word
- **Methodology:** Manual Functional Testing, Boundary Value Analysis, Negative Testing.
